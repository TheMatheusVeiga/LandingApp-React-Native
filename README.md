This is a simple project about coding techniques and new fetaures.

## Installation

OS X & Linux:

```sh
npm install expo-cli --global
```
```sh
expo init my-new-app
```

Windows:

```sh
npm install expo-cli --global
```
```sh
expo init my-new-app
```

## Development setup and Scripts

Avaliable scripts in the project.

```sh
npm run android
npm run ios
```
```sh
yarn start 
```

```sh
yarn build 
```
```sh
yarn test 
```
